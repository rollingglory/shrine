# Rolling Soft Loan

### Rolling Loan Initiative (RLI)

Rolling Soft Loan Initiative adalah fasilitas Rolling Glory untuk seluruh coaster dalam bentuk kesempatan untuk mengajukan pinjaman dengan bunga 0% (_soft loan_).
